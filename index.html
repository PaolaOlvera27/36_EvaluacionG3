Else-Sí no. Else exite gracias if, se ejecuta en caso de que if sea falsa, si no se cumplio la condicionante if entonces se ejecuta el codigo dentro de else

9) Qué son los objetos, cuál es su sintaxis y cómo puedes acceder a sus propiedades?:

Los Objetos son cualquier elemento que tenga propiedades las cuales uno asigne, esas propiedades son las que uno cree para determinado objeto, existen dos tipos de objeto literal o constructor, en el objeto literal es en el caso que solo sean propiedades de un solo objeto, en el constructor las propiedades se le pueden asignar a otros objetos

auto {
color:negro;
llantas: 4;
tipo:deportivo;
}

Para acceder a las propiedades es de la sig manera
auto.color
